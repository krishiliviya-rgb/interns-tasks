# 📄 AI-Based Resume Screening System

## 📌 Domain
Machine Learning

## 📖 Project Overview
This project is a Machine Learning-based Resume Screening System that ranks resumes according to their similarity with a given job description.

The system uses:
- TF-IDF Vectorization
- Cosine Similarity
- Skill Gap Detection

It helps recruiters automatically identify the most relevant candidates.

---

## 🚀 Features

✔ Resume Ranking based on job relevance  
✔ Match Score (in percentage)  
✔ Missing Skills Detection  
✔ Clean and modular code structure  

---

## 🛠 Technologies Used

- Python
- Scikit-learn
- Natural Language Processing (NLP)
- TF-IDF
- Cosine Similarity
